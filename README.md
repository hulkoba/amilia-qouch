# amilia-qouch

## Installation
### Install CouchDB
```
sudo apt-get install couchdb
```
Mac: download from [https://couchdb.apache.org/#download](https://couchdb.apache.org/#download)

#### Set up CORS
```
npm install -g add-cors-to-couchdb
// and run
add-cors-to-couchdb
```
#### Next
```
npm install
npm start
```
Have fun on [http://localhost:3000/](http://localhost:3000/)
